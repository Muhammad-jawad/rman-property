== Description ==

The Rman-Property-Plugin has been developed exclusively for Rentman Property. It facilitates the retrieval of property information from a designated account and seamlessly integrates it into your website using the [properties_list] shortcode.

*   rentman, rentman api, rman.co.uk

*   WordPress Version 6.4.2

*   Tested Upto Version 6.4.2


== Installation ==

* Upload Plugin

* Navigate to the "Settings" section and access the "Rentman Setting."

* Add the required API key to fetch the necessary data.

* Utilize the Google API to enable the display of Google Maps.

* Implement the form shortcode to enable a convenient popup for inquiries.
